# Sistema de Sanciones

El sistema de sanciones de Infernumcraft busca mantener el orden sin eliminar la posibilidad de corrección.

No todas las faltas merecen la misma respuesta. Una falta leve puede terminar en advertencia, mientras que hacks, duplicaciones o sabotajes graves pueden llevar directamente a expulsión definitiva.

## Principios

- Proporcionalidad.
- Registro de antecedentes.
- Reparación del daño cuando sea posible.
- Protección de la comunidad.
- Flexibilidad ante casos no previstos.

## Escala general

1. Advertencia.
2. Kick.
3. Ban temporal.
4. Expulsión definitiva.
