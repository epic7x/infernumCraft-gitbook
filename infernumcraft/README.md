# Infernumcraft

Infernumcraft es un servidor PvE de supervivencia extrema inspirado en experiencias tipo permadeath, pero adaptado a una temporada más compacta, intensa y progresiva.

La idea principal es simple: **sobrevivir no debe ser gratis**. Cada muerte, cada mala decisión, cada error de organización y cada descuido puede afectar el progreso individual o colectivo del servidor.

A diferencia de una survival común, Infernumcraft gira alrededor de tres pilares:

1. **Vidas limitadas.**
2. **Aumentos de dificultad progresivos.**
3. **Consecuencias compartidas para toda la comunidad.**

Esto significa que el servidor no está diseñado para jugarse como una survival casual, sino como una experiencia donde la preparación, la confianza, la cooperación y el juego limpio son fundamentales.

## Qué espera Infernumcraft de sus jugadores

Infernumcraft espera que cada jugador:

- Respete el reglamento.
- Juegue limpio.
- No busque ventajas externas.
- No arruine la experiencia de otros.
- Entienda que la dificultad es parte central del proyecto.
- Acepte que sus errores pueden tener consecuencias.

No se busca crear una comunidad perfecta, pero sí una comunidad donde todos entiendan que el servidor funciona mejor cuando existe respeto, responsabilidad y compromiso.
