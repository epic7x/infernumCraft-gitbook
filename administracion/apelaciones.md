# Apelaciones

Todo jugador sancionado puede apelar.

## Qué debe incluir una apelación

- Nombre de usuario.
- Sanción recibida.
- Fecha.
- Motivo por el que cree que debe revisarse.
- Pruebas o contexto adicional.

## Qué no hacer

- Insultar al Staff.
- Amenazar.
- Mentir.
- Enviar apelaciones repetidas.
- Crear presión pública para forzar una decisión.

## Resultado posible

Una apelación puede terminar en:

- Sanción mantenida.
- Reducción de sanción.
- Eliminación de sanción.
- Aumento de sanción si se descubre mentira o manipulación.
