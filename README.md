# Infernumcraft

**ESTA PAGINA ESTA SUJETA A CAMBIOS PERMANENTES QUE SERAN AVISADOS**


Bienvenido a la documentación oficial de **Infernumcraft**.

Infernumcraft es una experiencia de supervivencia extrema en Minecraft donde cada muerte importa, la dificultad aumenta progresivamente y las decisiones de los jugadores pueden afectar a toda la comunidad.

Esta wiki contiene:

- Explicación general del proyecto.
- Mecánicas principales.
- Reglamento oficial.
- Sistema de sanciones.
- Procedimientos administrativos.
- Lore del servidor.

> El objetivo de esta documentación es que cualquier jugador pueda entender qué se espera de él antes de entrar al servidor.
