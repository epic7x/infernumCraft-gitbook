# Mecánicas

Esta sección explica las mecánicas centrales de Infernumcraft.

Las mecánicas no reemplazan el reglamento, pero ayudan a entender por qué ciertas normas existen. Por ejemplo, debido a que las vidas son limitadas, causar la muerte intencional de otro jugador es una falta mucho más grave que en una survival común.

## Mecánicas principales

- Sistema de vidas.
- Death Train.
- Aumentos de dificultad.
- Conejos Vitales.
- Khrenos.

Estas mecánicas forman la identidad del servidor y deben ser respetadas por todos los participantes.
