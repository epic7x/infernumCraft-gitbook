# Sistema de Vidas

El sistema de vidas es una de las mecánicas más importantes de Infernumcraft.

Cada jugador cuenta con una cantidad limitada de vidas. Perderlas todas puede significar quedar fuera de la temporada, pasar a modo espectador o perder la capacidad de seguir participando activamente, según la configuración final del servidor.

## Propósito

El propósito del sistema de vidas es hacer que la supervivencia tenga peso real. No se busca castigar sin sentido, sino lograr que los jugadores midan mejor sus riesgos.

## Conductas esperadas

Los jugadores deben:

- Evitar muertes innecesarias.
- Prepararse antes de explorar zonas peligrosas.
- No usar bugs para recuperar vidas.
- No provocar muertes ajenas.
- Respetar las mecánicas oficiales de recuperación de vidas.

## Ejemplos de buen uso

- Un jugador espera a tener mejor equipo antes de entrar al Nether.
- Un equipo prepara pociones, comida y rutas de escape antes de explorar.
- Un jugador decide retirarse de una pelea peligrosa para no arriesgar su última vida.

## Ejemplos de abuso

- Provocar muertes intencionales para activar efectos.
- Usar bugs de resurrección.
- Coordinarse para manipular el sistema de vidas.
- Ocultar errores que permitan recuperar vidas ilegalmente.

Los abusos relacionados con vidas serán considerados faltas graves.
